# Javascript-masterclass
A chapter-wise JavaScript learning repository covering variables, data types, operators, conditions, loops, functions, arrays, objects, asynchronous JavaScript, and DOM fundamentals with practical examples.

# 💻 Installation


Navigate to the project:

```bash
cd javascript-masterclass
```

Run examples:

```bash
node filename.js
```
